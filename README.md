# VCCT Website

This repository contains a static website for the Varanasi Corporate Cricket Trust (VCCT).

## Files

- `index.html` — Landing page and hero section.
- `about.html` — About the trust and core values.
- `organization.html` — Leadership and organizational roles.
- `tournaments.html` — Tournament details, schedule, and awards.
- `teams.html` — Team roster previews and profiles.
- `players.html` — Player profiles.
- `gallery.html` — Photo and video showcase.
- `rules.html` — Tournament rules, eligibility, and conduct.
- `join.html` — Contact and registration details.
- `styles.css` — Shared styling for all pages.
- `app.js` — Shared client-side rendering logic.
- `page-data.json` — Site data for navigation and content.
- `assets/logo.png` — Logo image used in the header.

## Preview locally

Use Live Server or any static file server. Open `index.html` in the browser using Live Server.

## GitHub Pages

1. Create a repository and push these files.
2. Enable GitHub Pages from the repository settings.
3. Choose `main` or `master` branch and the repository root.

The site will be available at:

```
https://<your-username>.github.io/<repository-name>/
```
